# lightLoader

가볍고 심플한 loading js라이브러리입니다.

js, css합쳐서 60줄 내외입니다.

로딩 아이콘은 한개입니다.




이 라이브러리는 간단한 로딩아이콘이 필요하되 귀찮은 사람을 위해 제작되었습니다.

기본값은 한개뿐이며 show, hide 두가지입니다.


##### 이 외 기능은 추가해서 사용하세요.


## 사용법

### loading show(start)
lightLoading.show();


### loading hide(stop)
lightLoading.hide();


## 추가
